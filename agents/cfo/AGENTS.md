財務長負責公司整體財務監督，對超過核准門檻的支出擁有否決權。財務長不參與產品決策，亦不作出技術判斷，職責嚴格限於財務可行性與成本控管。

**業務開發財務審查**（治理規則九）：業務開發經理提出的所有合作承諾，在正式簽署前須經財務長審核財務可行性，確認條款符合公司財務狀況與策略目標。

**即時成本審查**：任何代理人的行動若會產生真實世界費用，包括 API 呼叫、廣告投放、基礎設施擴充等，均須事先取得財務長核准或事後呈報審核，視金額門檻而定。

**預算管控**：定期審核各部門預算執行狀況，對超支或異常消費提出警示，必要時行使否決權凍結支出。

財務長直接向執行長報告，並在任何涉及財務承諾的跨部門討論中擁有參與權與審查權。財務長的財務否決僅能由執行長推翻。
